

<h2>1. 자바</h2>
<p>
<h4>자바는 프로그램을 만들기 위해 사용하는 프로그래밍 언어이며, 보통 컴퓨터에게 명령하는 코드를 작성할 때 사용된다. </h4>
<h5>대표적인 특징으로는 <mark>객체 지향 언어</mark>이고, <mark>운영 체제에 독립적</mark>이란 점이 있다.</h5><br><br>
  
<h2>2. 변수와 연산자</h2
<p>
<h4>자바를 사용하기 위해 기본적으로 알아두는 개념이다.</h4><br>
<ul>
  <li><b>변수</b></li>
  <h5>변수란 변수는 데이터를 저장하기 위한 공간이다.</h5><br>
  <li><b>연산자</b></li>
  <ul> <br>
    <b><li>산술 연산자</li>
    <h5>+, -, *, /</h5>
    <li>증감 연산자</li>
    <h5>++, --</h5>
    <li>논리 연산자</li>
    <h5>&&, ||, !</h5>
    <li>대입 연산자</li>
    <h5>=, +=, -=, *=, /=, %=</h5>
    <li>비교 연산자</li>
    <h5>==. !=, >, <, >=, <= </h5> </b>
  </ul>
</ul> <br><br>

<h2>3. if문</h2>
<h4>if문은 조건을 판단하여 조건이 참일 때 실행하는 조건문이다.</h4> <br>
<h5>기본 구조 </h5>
<pre>
  if (조건식) {
    실행할 코드
}
</pre> <br>

  <h3><li>else와 else if</li><h3>
  <h5> else는 위의 if문이 모두 불만족할때 실행된다.</h5>
      <h5>else if는 복잡한 if문을 정리해 <mark>순서대로</mark> 조건 여러개를 나열할 수 있다. </h5> <br>
      <h5>예시</h5>
      <pre>
        if(조건) {
        }  
        //조건 1 만족 시 실행
        else if (조건2) {
        }  
        //조건 2 만족 시 실행
        else {
        }  
        //모두 불만족 시 실행
      </pre>
  
  <h3><li>switch</li></h3>
  <h5>switch문은 조건에 따라 맞는 값을 실행한다. <mark>단, break를 꼭 사용해야한다.</mark> </h5> <br> 
    
 <h3><li>삼항연산자</li></h3>
 <h5>참이라면 : 의 왼쪽을, 거짓이라면 오른쪽을 출력한다.</h5>
 <h5>예시</h5>
 <pre>
   int age = 15;
   String status = (age >= 18) ? "성인" : "미성년자"; <br>
   출력값 = 미성년자
 </pre>
</ul> <br> <br>

<h2>4. 반복문</h2>
<h4>반복문은 특정 조건이 만족되는 동안 같은 코드를 반복하는 문법이다.</h4> <br> <br>

<ul>
  <h3><li>while문</li></h3> <br>
    <ul>
    <li>복잡한 조건을 가진 반복문에 적합하다.</li>
    <li>do-while문은 먼저 한번 실행 후 조건을 체크하고 판단한다.
    <li>참고 : while(true)는 무한반복된다.</li></h5>

<h5>예시</h5> 
    <pre>
      int count = 1;
      while (count) = 1 + count;
      System.out.println(count); </pre>
      
      
  <br> <br> </ul>
  <ul>
  <h3><li>for문</li></h3> <br>
    <ul>
          <li>초기화, 체크, 반복 후 작업 등을 한번에 처리 가능해 편리하다.</li>
          <li>실무에선 대부분 for문을 사용한다고 한다.</li>
          <li>for문의 실행 순서는 <mark>초기식, 조건식, 코드, 증감식</mark> 순이다. 초기식은 1번만 실행한다.</li> </ul> <br> <br>

<hr>
<br> 
  <h4> <sup>+ </sup>중첩 반복문은 반복문 내에 반복문을 만들 수 있다.</h4>
    <h4> <sup>+ </sup>중첩해서 사용할시 <mark>내부와 외부의 곱</mark>으로 수행된다.</h4>
    <h4> <sup>+ </sup>반복문의 break로 while문을 빠져나갈 수 있다.</h4>
    <h4> <sup>+ </sup>반복문의 continue는 다시 조건문으로 돌아간다.</h4>
 <br> <br>

 <h2>5. 스코프와 형변환</h2>
 <p>
<ul>
  <li><b>스코프</b></li>
  <h5>스코프는 변수를 선언한 위치에 따라 분류되며, 메모리 낭비를 막기 위해 구분한다.</h5><br>
  <li><b>형변화</b></li>
   <br>
    <b><h5>형변화는 둘의 형태를 맞게 변경해주며 이에 의해 소수점 등이 날아가버릴 수도 있다.
      <b><mark>작은 숫자가 큰 숫자에 들어간다면 자동</mark>으로 옮겨지나, 반대의 경우 앞에 해당 꼴(---)을 기재해야한다.</b> </h5> <br> <br>
      <h5>예시</h5>
      <pre>
        double doublevalue = 1.5;
        int intValue = 0; //intValue = doubleValue는 컴파일 오류.
        intValue = (int) doubleValue; //처럼 바꿔줘야함.
      </pre>
      
</ul> <br><br>
